JavaScript Tutorial: Basic Array Methods

Arrays and their many methods offer JavaScript developers many tools to do a lot of really neat tricks, from simply adding or removing elements to dynamically creating HTML content. You'd probably waste more time making your own workaround when one may already exist as an array function!

Click here to see the Data Dictionary for a reference on the syntax.

Adding elements to an Array

If you want to add elements to an array, you can use:

  • Array.push(element) to add to the end of the array
  • Array.unshift(element) to add to the beginning of the array 
let array = [0, 1, 2];
                
array.push(3);
                
//array value: [0, 1, 2, 3]
let array = [0, 1, 2]; 
                
array.unshift(-1);
                
//array value: [-1, 0, 1, 2]

Removing elements from an Array

If you want to remove elements from an array, you can use:

  • Array.pop() to remove the last element from the array
  • Array.shift() to remove the first element from the array 
let array = [0, 1, 2]; 
                
array.pop();
                
//array value: [0, 1]
let array = [0, 1, 2]; 
                
array.shift();
                
//array value: [1, 2]

Splicing elements into an Array using the splice method

The Array.splice() method modifies the current array and has 3 parameters, the 1st of which is the only one that is mandatory. You can use splice() to remove elements from an array and even insert some in their place!

  • Array.splice(startingIndex [, nElementsToDelete, ...elementsToInsert])
    • startingIndex : int: start deleting elements at the given index of the array.
    • nElementsToDelete : int: delete up to n elements. Omitting this parameter will delete every element after the startingIndex.
    • ...elementsToInsert : any: accepts any number of elements of any type to insert at the given index. Separate additional elements after the first by a comma.

Using only the startingIndex parameter, which starts deleting at index 2 and so on.

let array = ['a', 'b', 'c', 'd']; 
                
array.splice(2);
                
//array value: ['a', 'b']

Using the startingIndex and nElementsToDelete parameter, which starts at index 1 and deletes 1 element.

let array = ['a', 'b', 'c', 'd']; 
                
array.splice(1, 1);
                
//array value: ['a', 'c', 'd']

Using the startingIndex, nElementsToDelete, and ...elementsToInsert parameters, which starts at index 1, deletes 1 element, and inserts 'B' at the startingIndex.

let array = ['a', 'b', 'c', 'd']; 
                
array.splice(1, 1, 'B');
                
//array value: ['a', 'B', 'c', 'd']

An example using the ...elementsToInsert parameter to insert multiple elements.

let array = ['a', 'b', 'c']; 
                
array.splice(1, 1, 'B', 2, 'D');
                
//array value: ['a', 'B', 2, 'D', 'c']

Creating an Array or Sub-Array using the slice method

The Array.slice() method creates a new array from the original array and has 2 parameters, both of which are optional. You can omit both parameters to create a copy of an array if you want to keep the original intact!

  • Array.slice([startingIndex, endingIndex])
    • startingIndex : int: starts copying elements from the given index.
    • endingIndex : int: stops copying elements at the given index, exclusive ( i.e. it does not copy the value at the endingIndex). Omitting this parameter will copy every element after the startingIndex.

Omitting parameters to make a copy of the array.

let array = ['a', 'b', 'c']; 
                
let slicedArray = array.slice();
                
//array value: ['a', 'b', 'c']
                
//slicedArray value: ['a', 'b', 'c']

Using the startingIndex parameter to make a copy of the array starting from index 1.

let array = ['a', 'b', 'c']; 
                
let slicedArray = array.slice(1);
                
//array value: ['a', 'b', 'c']
                
//slicedArray value: ['b', 'c']

Using the startingIndex and endingIndex parameters to make a copy of the array starting from index 1 and stopping at index 3.

let array = ['a', 'b', 'c', 'd']; 
                
let slicedArray = array.slice(1, 3);
                
//array value: ['a', 'b', 'c']
                
//slicedArray value: ['b', 'c']

Creating an Array using the concat method

The Array.concat() method creates a copy of the original array and has 1 parameter that accepts multiple elements. You can omit the parameter to create a copy of an array if you want to keep the original intact!

  • Array.concat([...elements])
    • ...elements : any: accepts any number of elements of any type. Separate additional elements after the first by a comma.

Omitting parameters to make a copy of the array.

let array = ['a', 'b', 'c']; 
                
let concatArray = array.concat();
                
//array value: ['a', 'b', 'c']
                
//concatArray value: ['a', 'b', 'c']

Using the ...elements parameter to make a new array from multiple elements. Array arguments are flattened into a single array.

let array = ['a', 'b', 'c'];
                
let concatArray = array.concat('d', ['e', 'f']); 
                
//array value: ['a', 'b', 'c']
                
//concatArray value: ['a', 'b', 'c', 'd', 'e', 'f']

Data Dictionary

Name Definition
element Refers to any data type, such as string, int, object, or any.
Parameter : element A parameter is the name of the value that gets passed into and used within a function, such as name in function setName(name). Parameters followed by a colon and a data type are constrained by that type, such as int, string, object, or any.
Argument An argument is sometimes used interchangeably with parameter, but an argument is the actual value that is passed into a function when it is called, such as "Susie" in setName("Susie").
function(mandatoryParameter) A function that contains a parameter outside of brackets is considered to be a mandatory parameter.
function([optionalParameter]) A function that contains a parameter inside brackets is considered to be an optional parameter.
function(...Arguments) A function that accepts any number of arguments, such as Array.splice().
